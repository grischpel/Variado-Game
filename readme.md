# Spielprojekt

## Anforderungen

Dieses Projekt benötigt einen lokalen Server zum Ausführen.

## Installation und Start

1. Stelle sicher, dass [Node.js](https://nodejs.org/) installiert ist.

2. Navigiere in das Projektverzeichnis:

3. Starte den Server mit:
```bash
npx serve
```

4. Öffne deinen Browser und navigate zur angezeigten URL (standardmäßig `http://localhost:3000`).

Das Spiel ist nun spielbereit!

## Hinweise

- `npx serve` erfordert keine globale Installation
- Der Server wird automatisch beendet mit `Ctrl + C`
